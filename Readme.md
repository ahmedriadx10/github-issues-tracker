# 1️⃣ What is the difference between var, let, and const?

Ans:
🔹 var is a function scope variable, where let and const are block scope variables
🔹var doesn't follow block scope rules, but let and const follow
🔹var, let, and const are all hoisted. var hoisted with an undefined value, but let and const stay on TDZ (Temporal Dead Zone)
🔹Using var and let to declare a variable can re-assign, but const doesn't

# 2️⃣ What is the spread operator (...)?

Ans: The spread operator is a JavaScript ES6 feature. By using the spread operator, we can easily spread all elements or properties anywhere

# 3️⃣ What is the difference between map(), filter(), and forEach()?

Ans: .map() .forEach(), and .filter() are all advanced array methods. They run once for every array element. .map() is used to apply an operation to every array element and returns the results as a new array. .forEach() is also similar to .map(), but .forEach() doesn't return anything. On the other hand, .filter ()method is used to filterout array elements depending on a condition. filter () returns filtered elements conditionally as an array

# 4️⃣ What is an arrow function?

Ans: An arrow function is a shorthand version of a function declaration. We can do almost all the work of function declaration with arrow functions.

# 5️⃣ What are template literals?

Ans: Template literals are an ES6 feature for creating dynamic or multiline strings.
